# 1-2-0 Chapter 2 の地図

> Claude Code を使い始める準備として、プランの契約と AI 利用ポリシー・データ取り扱いの確認を行う Chapter です。技術的な準備（プラン契約）を先に終え、その後で AI ツールを実務で使ううえで欠かせないポリシー確認を行います。

## セクション一覧

### 1-2-1 プランの選択と費用 ｜ 📖 読んで学ぶ

- 個人向けプラン（Free / Pro / Max）と Team / Enterprise / API の比較
- Pro と Max の違いは「使用量の上限」のみで機能は同じ、Pro生 への推奨プラン
- claude.ai でのアカウント作成・契約手順と Anthropic Console（API）アカウントとの違い

### 1-2-2 AI 利用ポリシーとデータ取り扱いの確認 ｜ 📖 読んで学ぶ

- 所属組織で確認すべき4つの項目（使用可否・コードの外部送信・機密情報・生成コードの扱い）
- Anthropic のデータ取り扱いポリシーとプライバシー設定
- テレメトリ／エラーレポートの正体と無効化方法（`DISABLE_TELEMETRY=1` など）
- Claude Code のセキュリティ機構（権限ベース・書き込み制限・サンドボックス）

## 📖 この Chapter の進め方

この Chapter もすべて概念セクションです。Claude Code を安心して使い始めるための「準備作業」を行います。まず本セクションでプラン契約を完了させ、続く [1-2-2 AI利用ポリシーとデータ取り扱いの確認](1-2-2_AI利用ポリシーとデータ取り扱いの確認.md) で AI 利用ポリシーとデータ取り扱いを確認します。
