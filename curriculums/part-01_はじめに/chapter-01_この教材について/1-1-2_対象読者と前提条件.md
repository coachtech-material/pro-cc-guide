# 1-1-2 対象読者と前提条件

## 🎯 このセクションで学ぶこと

- この教材の対象読者が誰かを理解し、自分に適しているか判断できる
- 学習を始めるために必要な前提知識を確認する
- 必要な環境（OS・ツール等）を把握する

まず対象読者を明確にし、次に前提知識をチェックリストで確認し、最後に必要な環境を把握します。

## 導入: この教材は誰のためのものか

この教材は、すべてのエンジニアに向けて書かれたものではありません。特定のスキルセットと状況にあるエンジニアが、最大の効果を得られるように設計されています。

### 🧠 先輩エンジニアはこう考える

> 「前提知識が完璧に身についていなくても大丈夫です。現場に入ったとき、自分の知識に不安を感じるのは誰でも同じです。大事なのは『わからないことに気づける』こと。チェックリストで不安な項目があったら、それは学ぶべきポイントが明確になったということ。Claude Code を使いながら、足りない部分は実践の中で補っていけます。」

対象読者は、プログラミングスクール COACHTECH の選抜試験を経てフリーランスエージェントに所属している**ジュニアエンジニア（Pro生）** です。企業に紹介され、副業・フリーランス・転職などの形で Laravel を使った実務開発に携わります。プログラミングの基礎力はありますが、実務経験はこれからという段階です。

次の条件に当てはまるなら、この教材はあなたのために書かれたものです。

- COACHTECH のカリキュラムを修了し、基礎的な Web アプリケーション開発ができる
- Laravel を使った開発経験がある（個人プロジェクトやスクール課題を含む）
- Git/GitHub を使ったチーム開発の基本的な流れを理解している
- これから実務に入る、または実務に入ったばかりで AI ツールを活用したい

## 前提知識チェックリスト

この教材では、以下の知識を前提としています。これらの基礎的な説明は省略しますが、Claude Code と組み合わせたときに注意が必要な点は丁寧に解説します。

各項目を確認し、自分の理解度を確かめてください。

### Laravel

- [ ] MVC アーキテクチャ（Model / View / Controller の役割分担）を理解している
- [ ] Eloquent ORM を使ったデータベース操作（CRUD、リレーション）ができる
- [ ] 認証（Fortify）と認可（Policy）の仕組みを理解している
- [ ] バリデーション（FormRequest）を実装できる
- [ ] ミドルウェアの役割と適用方法を理解している
- [ ] RESTful API の設計と実装の基本を理解している
- [ ] PHPUnit を使った基本的なテストを書ける

### フロントエンド

- [ ] HTML/CSS でページのレイアウトを構築できる
- [ ] Tailwind CSS の基本的なクラスを理解している

### データベース

- [ ] リレーショナルデータベースの設計（正規化、ER図）を理解している
- [ ] SQL で JOIN、サブクエリ、集計関数を使ったクエリを書ける

### 開発プロセス

- [ ] 要件定義 → DB/API 設計 → 実装 → テスト、という開発の一連の流れを理解している

### 開発環境・ツール

- [ ] ターミナル（コマンドライン）で基本的な操作（ディレクトリ移動、ファイル操作）ができる
- [ ] Docker Compose を使った開発環境の構築・起動ができる
- [ ] Git の基本操作（clone, branch, add, commit, push, pull）ができる
- [ ] GitHub でブランチ運用、PR 作成、コードレビューの流れを理解している

### AI 活用

- [ ] ChatGPT などの AI チャットツールを使ったことがある（少しでも可）

> 💡 AI コーディングツール（GitHub Copilot、Cursor、Claude Code 等）の経験は問いません。この教材でゼロから学びます。

チェックリストで不安な項目がいくつかあっても、問題ありません。すべてを完璧に理解している必要はなく、「聞いたことがある」「一度はやったことがある」レベルであれば十分です。特に Laravel の認証・認可やテストは、実務で触れながら理解を深めていく部分です。

ただし、ターミナルの基本操作と Git の基本操作は、Claude Code を使ううえで日常的に必要になります。これらに不安がある場合は、事前に復習しておくことを推奨します。

## 必要な環境

Claude Code を使うためには、いくつかの環境要件があります。学習を始める前に確認しておきましょう。具体的なインストール手順は Part 2 で扱いますので、ここでは「何が必要か」の把握だけで十分です。

### 対応 OS

Claude Code は macOS、Windows、Linux で動作します（2026年3月時点）。Windows では WSL（Windows Subsystem for Linux）を使う方法もあります。対応バージョンやセットアップの詳細は 2-1-1 で説明します。

### ハードウェア

- **RAM**: 4GB 以上
- **ネットワーク**: インターネット接続が必須（Claude Code はクラウド上の AI モデルと通信します）

### 必要なツール

- **ターミナル**: Bash、Zsh、PowerShell、または CMD
- **Git**: バージョン管理に使用（Claude Code が内部的にも利用します）
- **Docker Desktop**: Part 2 で使用する Laravel Sail の実行に必要です。受講中教材で導入済みのはずですが、未インストールの場合は事前に準備してください
- **テキストエディタ / IDE**: VS Code を推奨（Claude Code の拡張機能あり。ただし、この教材ではターミナルでの操作を基本とします）

### Anthropic アカウント

Claude Code を利用するには、Anthropic のアカウントと有料プランの契約が必要です。無料の Claude.ai アカウントでは Claude Code を利用できません。プランの詳細は「1-2-2 プランの選択と費用」で解説します。

### 対応国

Claude Code は Anthropic がサポートしている国・地域で利用できます。日本は対応しています。
