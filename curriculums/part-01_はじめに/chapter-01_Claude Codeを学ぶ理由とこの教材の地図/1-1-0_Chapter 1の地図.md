# 1-1-0 Chapter 1 の地図

> Claude Code を学ぶ意義・教材の全体像・必要な思考力・養う能力を理解し、自分がこの教材の対象であることを確認するための導入 Chapter です。「なぜ今これを学ぶのか」を腹落ちさせ、迷いなく学習を始められる状態を目指します。

## セクション一覧

| # | セクション | 学び方 | 概要 |
|---|---|---|---|
| 1-1-1 | Claude Code とは何か・教材の地図 | 📖 座学 | AI コーディングツールの普及状況、Claude Code の特徴、教材の 4 Part 構成 |
| 1-1-2 | AI との協働に求められる思考力 | 📖 座学 | 具体と抽象を行き来するピラミッドモデル、Laravel の注文機能を題材にした分解パターン比較 |
| 1-1-3 | この教材で養う3つの能力 | 📖 座学 | 使いこなす力（Input）・見極める力（Output）・学び続ける力（Time）の役割と相互関係 |
| 1-1-4 | 対象読者と前提知識 | 📖 座学 | COACHTECH Pro生 の定義、必要な前提知識、必要な環境 |

> 📝 **「学び方」マークについて**: この教材の各セクションは、Claude Code を実際に触るかどうかで2種類に分かれます。各 Chapter 冒頭の地図ページで毎回このマークを参照して、自分が今からどんな時間の使い方をするかを把握してください。
>
> | 学び方 | 意味 |
> |---|---|
> | 📖 **座学** | 概念を読んで理解するセクション。Claude Code は触らず、知識のインプットに集中する |
> | 🏃 **実践** | 概念を学びながら、または学んだあとに、自分のプロジェクトで Claude Code を実際に触るセクション |

## 📖 この Chapter の進め方

この Chapter はすべて座学です。手を動かす作業はありません。読みながら「自分がこの教材で何を得られるか」を整理してください。最初に教材全体の地図（このセクション）を頭に入れておくと、以降のセクションで「今やっていることが全体のどこに位置するか」が常に把握できます。
