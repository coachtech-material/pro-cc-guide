## 導入: この教材は誰のためのものか

この教材は、すべてのエンジニアに向けて書かれたものではありません。特定のスキルセットと状況にあるエンジニアが、最大の効果を得られるように設計されています。

### 🧠 先輩エンジニアはこう考える

> 前提知識が完璧に身についていなくても大丈夫です。現場に入ったとき、自分の知識に不安を感じるのは誰でも同じです。大事なのは「わからないことに気づける」こと。一覧で不安な項目があったら、それは学ぶべきポイントが明確になったということ。Claude Code を使いながら、足りない部分は実践の中で補っていけます。

対象読者は、プログラミングスクール COACHTECH で学び、Laravel を使った開発に携わる**ジュニアエンジニア**です。プログラミングの基礎力はありますが、実務経験はこれからという段階です。

次の条件に当てはまるなら、この教材はあなたのために書かれたものです。

- COACHTECH のカリキュラムを修了し、基礎的な Web アプリケーション開発ができる
- Laravel を使った開発経験がある（個人プロジェクトやスクール課題を含む）
- Git/GitHub を使ったチーム開発の基本的な流れを理解している
- これから実務に入る、または実務に入ったばかりで AI ツールを活用したい

## 前提知識

この教材では、以下の知識を前提としています。これらの基礎的な説明は省略しますが、Claude Code と組み合わせたときに注意が必要な点は丁寧に解説します。各分野で「聞いたことがある」「一度はやったことがある」レベルであれば十分です。

| 分野 | 想定する理解レベル |
|---|---|
| **Laravel** | MVC、Eloquent ORM（CRUD・リレーション）、認証（Fortify）と認可（Policy）、バリデーション（FormRequest）、ミドルウェア、RESTful API、PHPUnit による基本的なテスト |
| **フロントエンド** | HTML/CSS でレイアウトを構築できる。Tailwind CSS の基本的なクラスを理解している |
| **データベース** | リレーショナルデータベースの設計（正規化・ER 図）、SQL（JOIN、サブクエリ、集計関数） |
| **開発プロセス** | 要件定義 → DB/API 設計 → 実装 → テスト、という開発の一連の流れ |
| **ターミナル** | ディレクトリ移動、ファイル操作、コマンド実行などの基本操作 |
| **Docker Compose** | 開発環境の構築・起動・停止 |
| **Git / GitHub** | clone, branch, add, commit, push, pull の基本操作。ブランチ運用、PR 作成、コードレビューの流れ |
| **AI 活用** | ChatGPT などの AI チャットツールを使ったことがある（少しでも可） |

> 💡 AI コーディングツール（GitHub Copilot、Cursor、Claude Code 等）の経験は問いません。この教材でゼロから学びます。

すべての分野を完璧に理解している必要はありません。特に Laravel の認証・認可やテストは、実務で触れながら理解を深めていく部分です。

> ⚠️ ただし、**ターミナルの基本操作と Git の基本操作** は、Claude Code を使ううえで日常的に必要になります。これらに不安がある場合は、事前に復習しておくことを推奨します。

## 必要な環境

Claude Code を使うためには、いくつかの環境要件があります。学習を始める前に確認しておきましょう。具体的なインストール手順は Part 2 で扱いますので、ここでは「何が必要か」の把握だけで十分です。

### 対応 OS

Claude Code は macOS、Windows、Linux で動作します（2026年3月時点）。Windows では WSL（Windows Subsystem for Linux）を使う方法もあります。対応バージョンやセットアップの詳細は [2-1-1 インストールと認証](../../part-02_Claude%20Codeの基礎/chapter-01_セットアップ/2-1-1_インストールと認証.md) で説明します。

### ハードウェア

- **RAM**: 4GB 以上
- **ネットワーク**: インターネット接続が必須（Claude Code はクラウド上の AI モデルと通信します）

### 必要なツール

| ツール | 用途 |
|---|---|
| ターミナル | Bash、Zsh、PowerShell、または CMD |
| Git | バージョン管理に使用（Claude Code が内部的にも利用します） |
| Docker Desktop | Part 2 で使用する Laravel Sail の実行に必要。受講中教材で導入済みのはずですが、未インストールの場合は事前に準備してください |
| テキストエディタ / IDE | VS Code を推奨（Claude Code の拡張機能あり。ただし、この教材ではターミナルでの操作を基本とします） |

### Anthropic アカウント

Claude Code を利用するには、Anthropic のアカウントと有料プランの契約が必要です。無料の Claude.ai アカウントでは Claude Code を利用できません。プランの詳細は次の Chapter「[1-2-1 プランの選択と費用](../chapter-02_プラン契約とAI利用ポリシー/1-2-1_プランの選択と費用.md)」で解説します。

### 対応国

Claude Code は Anthropic がサポートしている国・地域で利用できます。日本は対応しています。
