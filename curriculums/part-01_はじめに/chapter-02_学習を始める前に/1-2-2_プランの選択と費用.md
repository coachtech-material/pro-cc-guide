# 1-2-2 プランの選択と費用

## 🎯 このセクションで学ぶこと

- Claude Code が使えるプランの種類と違いを把握する
- 自分に適したプランを選択できる
- 契約の手順を理解する

---

## Claude Code を使うにはプランの契約が必要

Claude Code は、無料の Claude.ai アカウントでは利用できません。利用するには、有料プランの契約または Anthropic Console（API）のアカウントが必要です。

「無料で試してから決めたい」と思うかもしれませんが、Claude Code については無料枠が提供されていません。この教材を進めるうえでも有料プランが前提となりますので、このセクションで各プランの違いを理解し、自分に適したプランを選択してください。

---

## プランの比較

Claude Code が利用できるプランを比較します（2026年3月時点）。

### 個人向けプラン

| プラン | 月額料金 | Claude Code | 使用量 |
|---|---|---|---|
| **Free** | 無料 | 利用不可 | ― |
| **Pro** | $20/月 | 利用可能 | 基本の使用量 |
| **Max 5x** | $100/月 | 利用可能 | Pro の5倍 |
| **Max 20x** | $200/月 | 利用可能 | Pro の20倍 |

### チーム・法人向けプラン

| プラン | 月額料金 | Claude Code | 備考 |
|---|---|---|---|
| **Team** | $25〜$150/ユーザー/月 | シートタイプによる | 最低5ユーザー。Premium シートで利用可能 |
| **Enterprise** | カスタム価格 | 利用可能 | 組織規模に応じた個別見積り |

> 📝 Team プランには Standard シート（$25/ユーザー/月）と Premium シート（$150/ユーザー/月）があります。Claude Code を利用するには Premium シートが必要です。公式ドキュメントでは「Teams」として一括で案内されていますが、シートタイプによって利用可能な機能が異なります（2026年3月時点）。

### API（従量課金）

Anthropic Console のアカウントを作成し、API キーを使って Claude Code を利用する方法もあります。使った分だけ課金される従量課金制で、モデルごとに料金が異なります。

> 📝 API での利用は、サブスクリプションプランとは異なる課金体系です。使い方によっては安くなることもありますが、使用量の管理が必要です。この教材では、サブスクリプションプラン（Pro または Max）での利用を前提とします。

---

## Pro プランと Max プランの違い

Claude Code が利用可能な個人向けプランは、Pro と Max の2種類です。機能面での違いはありません。違いは**使用量の上限** です。

### 使用量とは

Claude Code は、あなたの指示を処理するたびに AI モデルとの通信を行います。この通信量（トークン数）に応じて使用量が消費されます。Claude Code はエージェント型ツールのため、1つの指示に対して複数回の通信が発生します。たとえば「テストを書いて実行して」という指示では、コードベースの探索、テストコードの生成、テストの実行、結果の確認、必要に応じた修正と、複数のステップでトークンが消費されます。

### Pro プランの使用量

Pro プランでは、一定の使用量を超えるとレート制限がかかり、応答速度が低下したり、一時的に利用できなくなったりします。日常的なコードの質問や小規模な修正であれば十分ですが、Claude Code を頻繁に使って本格的に開発を進めると、制限に達することがあります。

### Max プランの使用量

Max プランは Pro の5倍（Max 5x: $100/月）または20倍（Max 20x: $200/月）の使用量が利用できます。Claude Code を日常的な開発ツールとして使い込む場合は、Max プランの方がストレスなく作業を進められます。

---

## Pro生へのおすすめ

この教材を進めるうえでの推奨プランは **Max プラン** 以上（Max 5x: $100/月〜）です。

Claude Code はエージェント型のツールであるため、チャット型 AI と比較して使用量の消費が大きくなります。教材のハンズオンでは、機能の試行錯誤や生成コードの修正で繰り返し Claude Code に指示を出す場面が多くあります。Pro プランの使用量では、学習の途中でレート制限にかかり、続きは翌日まで待つ、というストレスフルな状況になりかねません。

Max プランであれば、教材の学習に加えて実務での利用も余裕を持ってカバーできます。

Pro プランと Max プランの**体感的な違い** は、主にタスクの規模で現れます。1ファイルの修正やちょっとした質問なら Pro で十分ですが、「複数ファイルにまたがるリファクタリング」や「テストを書いて実行して修正するまで一気に」といった、Claude Code が何度もやり取りを繰り返すタスクでは Pro の使用量が急速に消費されます。学習中は試行錯誤で使用量が増えやすいため、「今日はもう使えない」というストレスが起きやすいのも Pro の注意点です。

ただし、月額 $100 は決して安い金額ではありません。まずは **Pro プラン（$20/月）** で始めて、使用量が足りないと感じたタイミングで Max にアップグレードする、という進め方でも問題ありません。

> 🔑 **最低条件: Pro プラン以上が必須です。** Free プランでは Claude Code を利用できないため、この教材を進めることができません。

| 状況 | おすすめプラン |
|---|---|
| 費用を抑えたい。制限がかかっても待てる | Pro（$20/月） |
| 学習と実務を並行して進めたい | Max 5x（$100/月）|
| Claude Code を主力ツールとして使い倒したい | Max 20x（$200/月） |

> 💡 企業に紹介された先で、チームとして Claude Code を利用する場合は、Team Premium プランや Enterprise プランが選択肢になります。その場合はチームの方針に従ってください。個人での学習には個人向けプランを使用します。

## 契約の手順

プランの契約は、Claude の Web サイトから行います。

### Step 1: アカウントの作成

[claude.ai](https://claude.ai) にアクセスし、アカウントを作成します。メールアドレスまたは Google アカウントで登録できます。

### Step 2: プランの選択

アカウント作成後、[claude.ai/pricing](https://claude.ai/pricing) からプランを選択します。Pro プランまたは Max プランの「Subscribe」ボタンをクリックし、支払い情報を入力します。

### Step 3: 契約の完了

支払い情報の入力が完了すると、すぐにプランが有効になります。Claude Code のインストールと認証は Part 2 で行いますので、ここではプランの契約まで完了していれば十分です。

> ⚠️ **よくある疑問**: 「API アカウント（Anthropic Console）と Claude.ai のアカウントは別物ですか？」
>
> はい、別物です。claude.ai のアカウント（Pro / Max プラン）と、Anthropic Console のアカウント（API 利用）は別のシステムです。この教材では claude.ai のアカウント（Pro または Max プラン）を使用します。Anthropic Console のアカウントは必要ありません。
