# 1-2-2 プランの選択と費用

## 概要

セクション 1-1-2 で触れたように、Claude Code の利用には有料プランまたは API アカウントが必要です。無料の Free プランでは Claude Code を利用できません。

月額 $20〜100 は決して小さな金額ではありません。ただ、実務で使い始めると、手作業で数時間かかっていた実装が数分で形になる場面を何度も経験します。キャリアの初期に開発スピードと品質を底上げする投資として、十分に元が取れるツールです。

このセクションでは、利用可能なプランの種類と機能差を整理し、皆さんに適したプランの選び方、契約手順、使用量の管理方法を説明します。

## プランの種類と機能差

### Claude のプラン体系

Claude には複数のプランがあり、Claude Code を利用できるのは以下のプランです。

| プラン | 料金（目安） | 対象 | Claude Code |
|---|---|---|---|
| Free | 無料 | 個人（お試し） | 利用不可 |
| Pro | $20/月 | 個人 | 利用可能 |
| Max | $100/月（5x）または $200/月（20x） | 個人（大量利用） | 利用可能 |
| Teams | $30/ユーザー/月（年払い） | チーム・企業 | 利用可能 |
| Enterprise | カスタム料金 | 大企業 | 利用可能 |
| Console（API） | 従量課金 | 開発者・企業 | 利用可能 |

⚠️ 上記の料金は 2026年3月時点の情報です。最新の料金は [claude.com/pricing](https://claude.com/pricing) で確認してください。

### 各プランの料金と Claude Code での機能差

**Pro プラン（$20/月）**

個人利用の基本プランです。Claude Code を含む Claude の主要機能を利用できます。この教材を進めるにあたっての最低条件です。

Claude Code の利用量に上限があり、集中的にコードを書く日には上限に達することがあります。上限に達すると、一定時間待つか、Max プランへのアップグレードが必要になります。

**Max プラン（$100/月 または $200/月）**

Pro プランの拡張版で、より多くの利用量が割り当てられます。$100/月プランは Pro の5倍、$200/月プランは Pro の20倍の利用量です。

この教材を集中的に進める場合や、実務で日常的に Claude Code を使う場合に推奨するプランです。利用量を気にせずにハンズオンに取り組めるため、学習効率が大きく向上します。

**Teams プラン（$30/ユーザー/月）**

チームでの利用を想定したプランです。管理者がメンバーのアクセス権やポリシーを管理でき、Managed settings による組織的な設定制御が可能です。データ訓練ポリシーが Commercial（Anthropic はモデル訓練にデータを使用しない）になるため、企業の業務コードを扱う場合に適しています。

**Enterprise プラン**

大企業向けのカスタムプランです。SSO（シングルサインオン）、監査ログ、Zero Data Retention（データ保持なし）などの高度なセキュリティ機能が含まれます。

**Console（API）**

従量課金で Claude のモデルに直接アクセスするプランです。使用したトークン数に応じて課金されます。Claude Code からも Console アカウントでログインして利用できます。

API の仕組みや Console の使い方は、この教材では詳しく扱いません。関心がある方は [Anthropic Console](https://console.anthropic.com/) を参照してください。

### サードパーティプロバイダー経由の利用

Anthropic のプランに加えて、以下のクラウドプロバイダー経由でも Claude Code を利用できます。

- **Amazon Bedrock**
- **Google Vertex AI**
- **Microsoft Foundry**

企業がこれらのクラウドサービスを契約している場合、既存のクラウド契約内で Claude Code を利用できます。API キーの設定方法はプロバイダーごとに異なります。企業のインフラチームに確認してください。

💡 この教材では、Anthropic の Pro または Max プランでの利用を前提に説明を進めます。サードパーティプロバイダー経由で利用する場合も、Claude Code の機能は基本的に同じです。

## プランの選び方とコスト管理

### この教材での推奨プラン

この教材を進めるにあたって、以下のようにプランを選択することを推奨します。

- **最低条件**: Pro プラン（$20/月）。Claude Code の全機能を利用できますが、利用量の上限に達する可能性がある
- **推奨**: Max プラン（$100/月の 5x プラン）。利用量を気にせずハンズオンに集中できる
- **企業が契約している場合**: 企業のプラン（Teams / Enterprise）を利用する。企業のポリシーに従うこと

まずは Pro プランで始め、利用量の上限に頻繁に達するようであれば Max プランへの変更を検討するのが現実的です。

### プランの契約手順

Claude Code を利用するには、まず Claude のアカウントを作成してプランを契約します。

1. [claude.ai](https://claude.ai) にアクセスし、アカウントを作成する（メールアドレスまたは Google アカウント）
2. ログイン後、設定画面からプランを選択する（Pro または Max）
3. 支払い方法（クレジットカード）を登録し、サブスクリプションを開始する

プランの契約が完了したら、次の Part 2 の最初のセクションで Claude Code のインストールと認証を行います。

💡 Console（API）で利用する場合は、[console.anthropic.com](https://console.anthropic.com/) でアカウントを作成し、クレジットを事前にチャージします。Claude Code からのログイン時に Console アカウントを選択できます。

### 使用量の管理

Claude Code の利用量は、プランに応じた上限の範囲内で管理されます。

**Pro / Max プラン** の場合、利用量の上限はウィンドウ制（一定時間内の利用量）で管理されます。上限に近づくと Claude Code 上に警告が表示されます。上限に達した場合は、ウィンドウがリセットされるまで待つか、プランをアップグレードします。

**Console（API）** の場合、利用量はトークン数に基づいて従量課金されます。Console のダッシュボードで費用をリアルタイムに確認できます。予算上限を設定することもできるため、想定外の費用が発生するリスクを抑えられます。

⚠️ Claude Code は1回のプロンプトで多くのトークンを消費することがあります。特に、大きなコードベースを読み込む場面やコード生成量が多い場面では消費量が増えます。Max プランを利用している場合でも、利用状況を時々確認する習慣をつけておくとよいでしょう。

## まとめ

- Claude Code の利用には Pro 以上のプラン、または Console（API）アカウントが必要。Free プランでは利用不可
- この教材では Pro プラン（$20/月）を最低条件、Max プラン（$100/月 5x）を推奨とする
- 企業が Teams / Enterprise を契約している場合は、企業のプランを利用する
- Amazon Bedrock、Google Vertex AI、Microsoft Foundry 経由でも利用可能
- 利用量の上限を意識し、必要に応じてプランの変更を検討する

この Chapter で、Claude Code を安心して使い始めるための準備が整いました。次の Part では、いよいよ Claude Code をインストールし、実際に手を動かしていきます。
