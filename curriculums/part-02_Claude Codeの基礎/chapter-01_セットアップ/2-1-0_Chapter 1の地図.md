# 2-1-0 Chapter 1 の地図

> Claude Code を使いこなすには、まずインストールと初期設定を確実に済ませる必要があります。この Chapter では、インストールから認証、権限設定、モデル選択、そしてプロジェクトの土台作りまでを一気に進めます。すべてのセクションで手を動かしながら学ぶ構成です。

| セクション | 学び方 | 内容 |
|---|---|---|
| 2-1-1 | 📖🏃 読んで手を動かす | インストールと認証。cc-practice を作成して最初のやり取りを体験する |
| 2-1-2 | 📖🏃 読んで手を動かす | 権限モデルと権限モードを理解し、安全に使う設定を行う |
| 2-1-3 | 📖🏃 読んで手を動かす | モデルの種類とコスト管理の方法を把握する |
| 2-1-4 | 📖🏃 読んで手を動かす | CLAUDE.md でプロジェクトの土台を作り、最初のコード生成を体験する |

## 📖 この Chapter の進め方

4つのセクションすべてが混合型（概念を学びながらすぐに手を動かす）です。最初のセクションで Laravel Sail プロジェクト「cc-practice」を作成し、以降のセクションでもこのプロジェクトを使い続けます。Part 2 を通じて cc-practice にさまざまな設定やコードを追加していき、Chapter 2-3 の終わりには Claude Code の主要機能を一通り組み込んだプロジェクトが完成します。上から順に進めてください。
