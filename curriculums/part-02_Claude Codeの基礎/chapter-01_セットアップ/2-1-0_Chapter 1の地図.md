# 2-1-0 Chapter 1 の地図

> Claude Code を使いこなすには、まずインストールと初期設定を確実に済ませる必要があります。この Chapter では、インストールから認証、権限設定、モデル選択、そしてプロジェクトの土台作りまでを一気に進めます。すべてのセクションで手を動かしながら学ぶ構成です。

## セクション一覧

### 2-1-1 インストールと認証 ｜ 📖🏃 読んで手を動かす

- インストール方法の選択（ネイティブ推奨、Homebrew、npm 非推奨）
- 初回起動時のブラウザベース認証フロー
- Laravel Sail プロジェクト「cc-practice」の作成と最初のやり取り

### 2-1-2 権限とセキュリティ ｜ 📖🏃 読んで手を動かす

- 3つの操作カテゴリ（読み取り・Bash・ファイル変更）と許可の仕組み
- 5つの権限モード（Default / Accept Edits / Plan / Don't Ask / Bypass Permissions）
- `.claude/settings.json` での権限ルール定義と機密情報の取り扱い

### 2-1-3 モデル選択とコスト管理 ｜ 📖🏃 読んで手を動かす

- 3つのモデル（Sonnet・Opus・Haiku）の特性と使い分け
- エフォートレベル（low / medium / high / max）による思考の深さ調整
- `/stats`・`/cost` での使用量確認とコスト節約のコツ

### 2-1-4 CLAUDE.md とプロジェクト作成 ｜ 📖🏃 読んで手を動かす

- CLAUDE.md の役割と配置場所（ユーザー全体・プロジェクト・サブディレクトリ）
- `/init` コマンドによる自動生成と `.claude/rules/` への分割
- 自動メモリ機能と見極めチェック（正しさ・品質・安全性）の初体験

## 📖 この Chapter の進め方

4つのセクションすべてが混合型（概念を学びながらすぐに手を動かす）です。最初のセクションで Laravel Sail プロジェクト「cc-practice」を作成し、以降のセクションでもこのプロジェクトを使い続けます。Part 2 を通じて cc-practice にさまざまな設定やコードを追加していき、[Chapter 2-3「機能を使いこなす」](../chapter-03_機能を使いこなす/2-3-0_Chapter%203の地図.md) の終わりには Claude Code の主要機能を一通り組み込んだプロジェクトが完成します。上から順に進めてください。
