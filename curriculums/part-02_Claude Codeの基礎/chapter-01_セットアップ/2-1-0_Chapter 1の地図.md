# 2-1-0 Chapter 1 の地図

> Claude Code を使いこなすには、まずインストールと初期設定を確実に済ませる必要があります。この Chapter では、インストールから認証、権限設定、モデル選択、そしてプロジェクトの土台作りまでを一気に進めます。すべてのセクションで手を動かしながら学ぶ構成です。

## セクション一覧

| # | セクション | 学び方 | 概要 |
|---|---|---|---|
| 2-1-1 | インストールと認証 | 🏃 実践 | インストール方法の選択、ブラウザベース認証、Laravel Sail プロジェクト「cc-practice」の作成 |
| 2-1-2 | 権限とセキュリティ | 🏃 実践 | 3つの操作カテゴリ、5つの権限モード、`.claude/settings.json` での権限ルール定義 |
| 2-1-3 | モデル選択とコスト管理 | 🏃 実践 | Sonnet / Opus / Haiku の使い分け、エフォートレベル、`/stats`・`/cost` での使用量確認 |
| 2-1-4 | CLAUDE.md とプロジェクト作成 | 🏃 実践 | CLAUDE.md の役割と配置、`/init` と `.claude/rules/` 分割、自動メモリと見極めチェック |

## 📖 この Chapter の進め方

4つのセクションすべてが実践型（概念を学びながらすぐに手を動かす）です。最初のセクションで Laravel Sail プロジェクト「cc-practice」を作成し、以降のセクションでもこのプロジェクトを使い続けます。Part 2 を通じて cc-practice にさまざまな設定やコードを追加していき、Chapter 2-3「機能を使いこなす」の終わりには Claude Code の主要機能を一通り組み込んだプロジェクトが完成します。上から順に進めてください。
