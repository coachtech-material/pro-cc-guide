# 2-2-0 Chapter 2 の地図

> Claude Code を効果的に使うには、その仕組みを理解することが不可欠です。この Chapter では、Claude Code の動作原理、コンテキスト管理、そしてプロンプト設計という3つの「考え方」を学びます。Chapter 2-3「機能を使いこなす」で各機能を実践するための土台になる内容です。

## セクション一覧

| # | セクション | 学び方 | 概要 |
|---|---|---|---|
| 2-2-1 | エージェントループ | 🏃 実践 | 収集 → 実行 → 検証 の3フェーズ、ツールの5カテゴリ、ツール実行ログの読み方と Esc 中断 |
| 2-2-2 | コンテキストとセッション管理 | 🏃 実践 | コンテキストウィンドウと自動コンパクション、`/clear`・`/compact`・`/btw`、Rewind |
| 2-2-3 | プロンプト設計 | 🏃 実践 | 4つの設計原則、`@filename` と `ultrathink`、よくある失敗パターンと対処 |

## 📖 この Chapter の進め方

3つのセクションすべてが実践型（概念を学んでから cc-practice で体験する）です。概念を読んで理解した後、各セクションの末尾にある 🏃 実践で手を動かして体感します。Chapter 2-3「機能を使いこなす」の機能実践に入る前の土台になる内容です。上から順に進めてください。
