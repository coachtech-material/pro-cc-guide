# 4-1-0 Chapter 1 の地図

> Claude Code と AI ツールの進化に追従し続ける方法を理解し、教材で学んだことを振り返り、さらに先の活用領域を知る Chapter です。Part 3 で鍛えた「使いこなす力」と「見極める力」に、最後のピース「学び続ける力」を加えて、教材の締めくくりとします。

## セクション一覧

### 4-1-1 学び続けるサイクル ｜ 📖 読んで学ぶ

- Claude Code の更新ペースと安定性ラベル（Experimental / Research Preview）の見方
- 情報収集の優先順位（Changelog・公式ドキュメント・Anthropic Blog・コミュニティ）
- 検証環境の作り方（使い捨てプロジェクト・Worktree・`--bare` モード）
- 適用判断の3観点（安定性・チームへの影響・コストとメリット）

### 4-1-2 教材の総まとめ・次のステップ ｜ 📖 読んで学ぶ

- 教材で養った3つの能力の振り返り
- Claude Code との協働で「変わるもの（速度・範囲・学習効率）」と「変わらないもの（責任・設計判断・説明責任）」の区別
- ジュニアからミドルへ成長するための3つの振る舞い（なぜを問う・影響範囲を考える・判断を残す）

### 4-1-3 Claude Code の発展機能を知る ｜ 📖 読んで学ぶ

- ローカル環境拡張（Computer Use・Voice Dictation・Chrome 連携・Desktop アプリ）
- リモート・Web 利用（Claude Code on the Web・Remote Control）
- チーム・自動化（Agent Teams・Channels・Slack 連携・Agent SDK）

## 📖 この Chapter の進め方

Pro生のあなたは、Part 2・3 を通じて Claude Code の基礎と実践を学んできました。Part 4 はハンズオンのない読み物パートです。Part 3 までの実践を振り返りながら、「これから先、どうやって学び続けるか」を考えるための Chapter です。

4-1-1 で学び続けるための具体的なサイクルを学び、4-1-2 で教材全体を振り返り、4-1-3 でまだ知らない機能に触れます。順番に読み進めてください。
