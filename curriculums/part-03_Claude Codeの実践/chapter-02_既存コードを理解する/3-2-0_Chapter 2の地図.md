# 3-2-0 Chapter 2 の地図

> 既存プロジェクトに参画したとき、最初に求められるのは「コードを読んで理解すること」です。この Chapter では、コードを効率的に読むための認知科学に基づく方法論を学び、Claude Code を活用して CourseHub のコードを実際に読み解きます。

| セクション | 学び方 | 内容 |
|---|---|---|
| 3-2-1 コードリーディングの方法論 | 📖 読んで学ぶ | 認知科学に基づくコードリーディングの5ステップ、Claude Code の活用法、検証方法 |
| 3-2-2 知らない技術への向き合い方 | 📖 読んで学ぶ | 未知の技術に出会ったときの対処フロー、理解の深さの判断基準、心構え |
| 3-2-3 CourseHub のコードを読む | 📖🏃 読んで手を動かす | 5ステップに沿って CourseHub のコードを読み、Sub-agents や MCP で効率的に探索し、Claude の説明を自分の目で検証する |

## 📖 この Chapter の進め方

まず概念セクション（3-2-1）で、なぜコードが読めないのかを認知科学の視点から理解し、認知負荷を下げる5ステップの方法論を学びます。次の概念セクション（3-2-2）で、知らない技術に出会ったときの向き合い方を学びます。最後に混合セクション（3-2-3）で、CourseHub のコードを5ステップに沿って実際に読み、Sub-agents や MCP を使って大量のコードを効率的に探索し、Claude の説明を自分の目で検証する体験をします。
