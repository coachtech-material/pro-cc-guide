# 3-2-0 Chapter 2 の地図

> 既存プロジェクトに参画したとき、最初に求められるのは「コードを読んで理解すること」です。この Chapter では、コードを効率的に読むための認知科学に基づく方法論を学び、Claude Code を活用して CourseHub のコードを実際に読み解きます。

## セクション一覧

| # | セクション | 学び方 | 概要 |
|---|---|---|---|
| 3-2-1 | コードリーディングの方法論 | 📖 座学 | 認知負荷の2つの源泉、認知負荷を下げる5ステップ、Claude の説明を検証するサイクル |
| 3-2-2 | 知らない技術への向き合い方 | 📖 座学 | 4ステップで使い分ける理解の深さ、説明できないコードは受け入れない原則、Laravel パターンの地図 |
| 3-2-3 | CourseHub のコードを読む | 🏃 実践 | Sub-agents での並列探索と MCP での DB 確認、階層構造とロール認可、受講登録フローの追跡 |

## 📖 この Chapter の進め方

まず座学（3-2-1）で、なぜコードが読めないのかを認知科学の視点から理解し、認知負荷を下げる5ステップの方法論を学びます。次の座学（3-2-2）で、知らない技術に出会ったときの向き合い方を学びます。最後に実践（3-2-3）で、CourseHub のコードを5ステップに沿って実際に読み、Sub-agents や MCP を使って大量のコードを効率的に探索し、Claude の説明を自分の目で検証する体験をします。
