| # | セクション | 学び方 | 概要 |

## 📖 この Chapter の進め方

まず 3-5-1 でリファクタリングの方法論を学びます。「動作を変えずにコードを良くする」ためのフローと、Claude Code の改善提案を判断する基準を理解してください。次の 3-5-2 では、CourseHub のコードを実際に改善します。1つ目は Fat Controller の責務分離（コード構造の改善）、2つ目は N+1 クエリの解消（パフォーマンスの改善）です。改善の対象が違えばアプローチも変わることを体験します。
