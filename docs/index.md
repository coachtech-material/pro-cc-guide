## この教材で身につく3つの力

| 能力 | 内容 |
|---|---|
| **使いこなす力（Input）** | Claude Code の機能・設定・制約を把握し、目的に応じて的確に活用する力 |
| **見極める力（Output）** | 生成コードを正しさ・品質・安全性の3観点で検証し、責任を持って採用する力 |
| **学び続ける力（Time）** | ツール・技術の進化を追い、自ら試し、実務に適応し続ける力 |
