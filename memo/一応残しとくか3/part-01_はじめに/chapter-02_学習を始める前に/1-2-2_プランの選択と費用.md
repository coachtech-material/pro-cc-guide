# 1-2-2 プランの選択と費用

## 概要

Claude Code を利用するには、対応するプランの契約が必要です。無料プランでは利用できません。

このセクションでは、各プランの特徴と違いを整理し、皆さんに適したプランの選び方、契約手順、コスト管理の方法を説明します。

## プランの種類と機能差

### Claude のプラン体系

Claude のプランは、個人向けの **Consumer プラン**と、組織向けの **Commercial プラン**に大別されます。加えて、API を直接利用する **Console（API）** と、サードパーティプロバイダー経由の利用があります。

| 区分 | プラン | Claude Code の利用 |
|---|---|---|
| Consumer | Free | 利用不可 |
| Consumer | Pro | 利用可能 |
| Consumer | Max | 利用可能 |
| Commercial | Teams | 利用可能 |
| Commercial | Enterprise | 利用可能 |
| API | Console（従量課金） | 利用可能 |
| サードパーティ | Amazon Bedrock / Google Vertex AI / Microsoft Foundry | 利用可能 |

**Free プランでは Claude Code を利用できません**。学習を始めるには、Pro 以上のプランの契約が必要です。

### Consumer プランの比較（Free / Pro / Max）

皆さんが個人で契約する場合、選択肢となるのは Consumer プランです。以下の表で、Free・Pro・Max の違いを確認しましょう。

| 比較項目 | Free | Pro | Max 5x | Max 20x |
|---|---|---|---|---|
| 月額料金 | $0 | $20 | $100 | $200 |
| Claude Code の利用 | **不可** | 利用可能 | 利用可能 | 利用可能 |
| 利用量（対 Pro 比） | — | 標準 | 5倍 | 20倍 |
| claude.ai（Web チャット） | 利用可能 | 利用可能 | 利用可能 | 利用可能 |
| データ訓練 | ユーザーが選択可能 | ユーザーが選択可能 | ユーザーが選択可能 | ユーザーが選択可能 |

⚠️ 上記の料金は執筆時点（2026年3月）の情報です。最新の料金は [claude.com/pricing](https://claude.com/pricing) で必ず確認してください。

**Free プラン**は claude.ai の Web チャットのみ利用でき、Claude Code は使えません。学習を始めるには Pro 以上が必要です。

**Pro プラン**は Claude Code を含む主要機能を利用できる個人向けプランです。利用量には上限があり、上限に達すると一時的に利用が制限されます。

**Max プラン**は Pro プランの上位プランで、5倍（$100/月）と20倍（$200/月）の2つの利用量ティアがあります。Claude Code はコードベースの読み取りやファイル編集など1回の操作でも多くのトークンを消費するため、集中的に使う場合は Max プランの利用量の余裕が大きな差になります。

### Commercial プランとその他の選択肢

企業に所属して作業する場合、以下のプランが利用できる可能性があります。

**Teams プラン**（$30/ユーザー/月）は組織向けのプランです。チームメンバーの管理やアクセス制御の機能が追加されます。

**Enterprise プラン**は大規模組織向けで、SSO（シングルサインオン）、監査ログ、Zero Data Retention などの高度なセキュリティ・管理機能を備えています。料金は個別見積もりです。

**Console（API）** は Anthropic の API を直接利用する従量課金方式です。トークン（入出力の文字量に応じた単位）ごとに課金され、サブスクリプション契約は不要です。開発用途やカスタムツールの構築に適していますが、この教材の学習には Consumer プラン（Pro または Max）の方が適しています。

**サードパーティプロバイダー**（Amazon Bedrock / Google Vertex AI / Microsoft Foundry）経由で Claude Code を利用することもできます。企業がこれらのクラウドサービスを契約している場合、そちら経由で Claude Code を使うよう指示されることがあります。

💡 企業に所属して作業する場合、企業が Teams / Enterprise プランやサードパーティプロバイダーを契約していることがあります。自分でプランを契約する前に、企業側のアカウントが利用できないか確認しましょう。

## プランの選び方とコスト管理

### この教材での推奨

この教材を進めるにあたっての推奨プランは以下のとおりです。

- **最低条件**: Pro プラン（$20/月）
- **推奨**: Max プラン（$100/月〜）

Pro プランでも教材の内容はすべて実践できます。ただし、Pro プランでは学習の途中で利用量の上限に達し、待ち時間が発生する可能性があります。Max プランであれば利用量に余裕があるため、学習に集中しやすい環境が得られます。予算に余裕がある場合は Max 5x（$100/月）以上を検討してください。

### プランの契約手順

Claude のプランは、以下の手順で契約できます。

1. [claude.ai](https://claude.ai) にアクセスし、アカウントを作成する（Google アカウントまたはメールアドレスで登録）
2. ログイン後、設定画面またはプラン選択画面に移動する
3. 希望するプラン（Pro または Max）を選択する
4. 支払い情報（クレジットカード）を入力し、サブスクリプションを開始する

💡 Console（API）を利用する場合は [console.anthropic.com](https://console.anthropic.com) から別途アカウントを作成し、API キーを発行します。この教材では Consumer プラン（Pro / Max）の利用を前提として進めます。

### コストの目安と使用量の管理

Claude Code の利用コストは、プランの月額料金に含まれています（Consumer プランの場合）。従量課金ではなく、月額固定のサブスクリプションです。

ただし、各プランには利用量の上限があります。上限に達すると、次のリセットまで利用が制限されるか、速度が低下します。利用量の状況は Claude Code 内で確認できます。

コストを管理するためのポイントをいくつか紹介します。

- **まず Pro プランから始める**: 利用頻度を見極めてから Max プランへの変更を検討する
- **利用量を意識する**: 大きなコードベースを一度に読み込ませたり、試行錯誤を繰り返すと、利用量が急速に増える。Part 2 以降で効率的な使い方を学ぶ
- **月の途中でのプラン変更**: プランはいつでもアップグレード・ダウングレードが可能

## まとめ

- Claude Code の利用には Pro 以上のプランが必要（Free プランでは利用不可）
- Consumer（Pro / Max）、Commercial（Teams / Enterprise）、Console（API）、サードパーティプロバイダーの選択肢がある
- この教材では Pro プランを最低条件、Max プランを推奨とする
- 企業に所属する場合は、企業のアカウントが利用できないか先に確認する
- 各プランの最新料金は [claude.com/pricing](https://claude.com/pricing) で確認する
