# 1-2-2 プランの選択と費用

## 概要

前の Section では、AI ツールを業務で使う前に確認すべきポリシーやデータの取り扱いを学びました。この Section では、Claude Code を利用するために必要なプランの選択と費用について解説します。

この Section では、以下の順に解説します。

- Claude のプラン体系と Claude Code での機能差
- 自分に適したプランの選び方とコスト管理

## プランの種類と機能差

### Claude のプラン体系

Claude Code は **Free プランでは利用できません**。利用するには、以下のいずれかのプランまたはアカウントが必要です。

- **Pro**: 個人向けの有料プラン
- **Max**: 個人向けの上位プラン（使用量上限が大幅に拡大）
- **Team**: チーム向けプラン
- **Enterprise**: 企業向けプラン
- **Console（API）**: API 経由での利用（従量課金）

また、サードパーティのクラウドプロバイダー経由でも利用可能です。

- **Amazon Bedrock**
- **Google Vertex AI**
- **Microsoft Foundry**

サードパーティプロバイダー経由の場合、料金体系やデータポリシーはそれぞれのプロバイダーの規約に従います。この教材では、Anthropic の公式プラン（Pro / Max）を前提に説明を進めます。

### 各プランの料金と Claude Code での機能差

以下は執筆時点（2026年3月）の参考情報です。

⚠️ 料金やプラン内容は変更される可能性があります。最新の情報は [claude.ai/pricing](https://claude.ai/pricing) を確認してください。

| プラン | 月額（税抜） | Claude Code 利用 | 使用量上限 | 主な特徴 |
|---|---|---|---|---|
| Free | 無料 | ❌ 利用不可 | — | Claude Code は利用できない |
| Pro | $20 | ✅ | あり（標準） | 学習開始に十分な機能 |
| Max（5x） | $100 | ✅ | Pro の5倍 | 集中的な開発に適している |
| Max（20x） | $200 | ✅ | Pro の20倍 | 大規模な開発に適している |
| Team | $30/ユーザー | ✅ | あり | チーム管理機能、データ訓練なし |
| Enterprise | 要問合せ | ✅ | カスタム | SSO/SCIM、ZDR、監査ログ |
| Console（API） | 従量課金 | ✅ | なし（クレジット制） | 使った分だけ課金 |

💡 Pro プランと Max プランの「使用量上限」は、一定期間内に Claude Code で使用できるトークン量の上限です。上限に達すると、一時的に利用が制限されます。

## プランの選び方とコスト管理

### 最低条件（Pro）と推奨プラン（Max）

この教材では、**Pro プラン（$20/月）を最低条件**とし、**Max プランを推奨**します。

**Pro プランで学習を開始できる理由**

- Claude Code の全機能（コード生成、テスト作成、Git 連携など）が利用可能です
- 教材のハンズオンは Pro プランの使用量上限内で進められるよう設計しています

**Max プランを推奨する理由**

- 実務で Claude Code を本格的に活用すると、Pro プランの使用量上限に達することがあります
- Max プランでは使用量上限が大幅に拡大され、作業の中断を避けられます
- 集中的に学習や開発を行う時期には、一時的に Max プランにアップグレードすることも有効です

⚠️ 企業に所属している場合は、企業が Team / Enterprise プランを提供しているか確認してください。企業プランが利用できる場合は、そちらを優先してください（データ訓練ポリシーの観点からも安全です）。

### プランの契約手順

Claude Code を利用するには、まず [claude.ai](https://claude.ai) でアカウントを作成し、プランを契約します。

1. [claude.ai](https://claude.ai) にアクセスし、アカウントを作成する（メールアドレスまたは Google / Apple アカウントで登録）
2. ログイン後、設定画面からサブスクリプションを選択する
3. Pro または Max プランを選び、支払い情報を入力する
4. サブスクリプションが有効になったら、Claude Code のインストールに進められる

💡 Claude Code のインストール手順は Part 2（Chapter 2-1）で詳しく解説します。この Section ではプランの契約までを扱います。

### コストの目安と使用量の管理

**Pro / Max プランの場合**

サブスクリプション制のため、月額固定で利用できます。使用量上限に達した場合は、次のリセットまで利用が制限されます。

Claude Code 内で `/cost` コマンドを実行すると、現在のセッションでのトークン使用量とコストの目安を確認できます。

**Console（API）の場合**

従量課金制のため、使った分だけ課金されます。API キーの発行時にクレジットの上限を設定することで、意図しない高額請求を防げます。

**使い過ぎを防ぐポイント**

- `/cost` コマンドでセッションごとの使用量を定期的に確認する
- 大きなタスクを依頼する前に、タスクを分割して段階的に進める
- コンテキストが膨らみすぎた場合は `/clear` や `/compact` で整理する（Chapter 2-3 で詳しく学びます）

## まとめ

- Claude Code は Free プランでは利用できない。Pro 以上のプラン、または Console（API）アカウントが必要
- Pro / Max / Team / Enterprise / Console のほか、Amazon Bedrock / Google Vertex AI / Microsoft Foundry 経由でも利用可能
- この教材では Pro プラン（$20/月）を最低条件、Max プランを推奨とする
- 企業の Team / Enterprise プランが利用できる場合は、そちらを優先する
- `/cost` コマンドでセッションの使用量を確認し、コストを管理する
