# 1-1-2 対象読者と前提条件

## 概要

前の Section では、Claude Code を学ぶ意義と教材の目標を説明しました。この Section では、この教材がどのような読者を対象としているのか、学習を始めるために必要な技術スキルと開発環境を確認します。

自分のスキルレベルと照らし合わせながら、以下の順に確認します。

- この教材の対象読者
- 前提となる技術スキル
- 前提となる開発環境

## この教材の対象読者

この教材は、スクールを優秀な成績で卒業し、選抜試験を経てフリーランスエージェントに所属しているPro生の皆さんを対象としています。皆さんはこれから、企業に紹介されて副業・フリーランス・転職などの形で Laravel を使った実務開発に携わっていきます。

皆さんはすでに開発の基礎力を持っています。この教材は、その基礎力の上に Claude Code の活用スキルを積み重ね、実務での生産性と品質をさらに高めることを目指しています。

## 前提となる技術スキル

この教材を進めるにあたって、以下の技術スキルを前提とします。すべてのスキルが完璧である必要はありませんが、各項目の基本的な概念と操作を理解していることが求められます。

| スキル | 前提レベル |
|---|---|
| **ターミナル操作** | 基本コマンド（`cd`, `ls`, `mkdir`, `mv`, `rm` 等）を使ったファイル操作ができる |
| **Git / GitHub** | ブランチ運用、PR 作成、コンフリクト解決、Issue 駆動開発、コードレビューができる |
| **Docker** | Docker Compose を使った開発環境の構築・管理ができる |
| **HTML / CSS** | レスポンシブデザインを含む Web ページを作成できる。Tailwind CSS の基礎がある |
| **PHP** | 基本文法、関数、OOP の基礎（クラス、コンストラクタ）を理解している |
| **Laravel** | MVC、Eloquent ORM、認証（Fortify）、認可（Policy）、バリデーション、ミドルウェア、RESTful API 開発、PHPUnit によるテストを実装できる |
| **データベース** | RDB 設計（正規化、ER 図）、SQL（JOIN、サブクエリ、集計）を扱える |
| **開発プロセス** | 要件定義 → DB / API 設計 → 実装 → テストの一連の開発フローを経験している |

💡 AI コーディングツール（GitHub Copilot、Cursor 等）の使用経験は問いません。ChatGPT を少し触った程度の方を想定しています。Claude Code の使い方はゼロから丁寧に解説しますので、AI ツール初心者の方も安心して進められます。

## 前提となる開発環境

この教材を進めるために、以下の環境が必要です。

### OS

- **macOS**: メインのサポート対象です
- **Windows**: WSL（Windows Subsystem for Linux）経由での利用を前提とします

### エディタ

- **VS Code**: この教材では VS Code を前提とします。Claude Code はターミナルで動作しますが、VS Code のターミナル内で利用することを想定しています

### ソフトウェア

| ソフトウェア | 必要性 | 備考 |
|---|---|---|
| **Git** | 必須 | インストール済みであること |
| **Docker Desktop** | 必須 | Part 2 のハンズオンで Laravel Sail を使用するため、Docker Desktop がインストール済みであること |

⚠️ **Node.js は不要です。** Claude Code のインストールに Node.js や npm は必要ありません。Claude Code はネイティブインストール（`curl` / Homebrew / WinGet）が推奨されており、npm を使ったインストールは非推奨（deprecated）です。インストール手順は Part 2 で詳しく解説します。

### Claude アカウント

Claude Code を使うには Claude のアカウントとサブスクリプションが必要です。

- **最低条件**: Pro プラン（$20/月）
- **推奨**: Max プラン（より多くの使用量が含まれ、学習を進めやすい）
- **企業所属の場合**: 企業の AI ツール利用ポリシーとプランを確認してください

プランの詳細と選び方は Chapter 1-2 で説明します。

### ブラウザ

Claude Code の初回認証でブラウザが必要です。特定のブラウザの指定はありません。

## まとめ

- この教材は、Laravel を使った実務開発に携わる Pro生（ジュニアエンジニア）を対象としている
- ターミナル操作、Git/GitHub、Docker、PHP/Laravel、データベースの基礎スキルを前提とする
- AI コーディングツールの経験は不要で、ゼロから解説する
- 開発環境として macOS または WSL、VS Code、Docker Desktop が必要
- Node.js は不要（Claude Code はネイティブインストールが推奨）
- Claude アカウント（最低 Pro プラン）が必要

次の Section では、教材の全体構成と各 Part の概要を説明し、皆さんに適した学習の進め方を案内します。
