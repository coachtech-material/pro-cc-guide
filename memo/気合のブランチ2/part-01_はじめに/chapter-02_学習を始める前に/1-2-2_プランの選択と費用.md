# 1-2-2 プランの選択と費用

## 🎯 このセクションで学ぶこと

- Claude Code が利用できるプランの種類と違いを理解する
- 自分の状況に合ったプランを選択できる
- アカウント作成の流れを把握する

---

## プラン選びが最初のステップ

前のセクションで、AI ツールを使う上での確認事項を整理しました。確認が済んだら、いよいよ Claude Code を使い始める準備です。

Claude Code を利用するには、Anthropic の **有料アカウント** が必要です。無料プラン（Free）では Claude Code は利用できません。「どのプランを選べばいいのか」は、最初に迷うポイントの一つです。このセクションで選び方を整理しましょう。

---

## 🧠 先輩エンジニアの思考プロセス

自分が初めて Claude Code を使い始めたとき、正直プランの違いがよくわからなくて、とりあえず一番安いプランで始めました。結果的にそれで正解でした。最初から高いプランにする必要はありません。使い方がわかってきて「もっと使いたい」と思ったときにアップグレードすればいい。大事なのは、まず使い始めることです。

## プランの種類と比較

> 📝 以下の料金情報は 2026年3月時点のものです。最新の料金は [claude.com/pricing](https://claude.com/pricing) で確認してください。

### 個人向けプラン

| プラン | 月額料金 | Claude Code | 利用量の目安 |
|---|---|---|---|
| **Free** | $0 | 利用不可 | チャットのみ（基本的な利用） |
| **Pro** | $20/月（月払い）または年額 $200（$17/月相当） | 利用可能 | 日常的な開発作業に十分 |
| **Max 5x** | $100/月〜 | 利用可能 | Pro の5倍の利用量 |
| **Max 20x** | 上位プラン | 利用可能 | Pro の20倍の利用量 |

### チーム / 企業向けプラン

| プラン | 料金 | Claude Code | 特徴 |
|---|---|---|---|
| **Team** | $25/ユーザー/月（月払い）または $20/ユーザー/月（年払い） | 利用可能 | SSO、管理機能、デフォルトで学習利用なし |
| **Enterprise** | $20/シート（基本）+ 利用量に応じた従量課金 | 利用可能 | 監査ログ、SCIM、カスタムデータ保持、HIPAA 対応 |

## どのプランを選ぶべきか

### この教材の学習には Pro プランで十分

結論から言うと、この教材で学習を進める上では **Pro プラン（$20/月）** で十分です。

Pro プランでは、Claude Code の全機能が利用できます。利用量に上限はありますが、学習用途や日常的な開発作業であれば、上限に達することはほとんどありません。

### Pro プランを選ぶ理由

- **Claude Code の全機能が使える**: 機能の制限はありません。Pro でも Max でも、使える機能は同じです
- **コストが抑えられる**: 月 $20（約3,000円前後）で始められます。年払いにすると $17/月とさらにお得です
- **アップグレードはいつでもできる**: 使い込んで利用量が足りなくなったら、Max プランにアップグレードできます

### Max プランを検討するタイミング

以下のような状況になったら、Max プランへのアップグレードを検討してください。

- 1日に何時間も Claude Code を使う開発スタイルが定着した
- 利用量の上限に頻繁に達するようになった
- 大規模なリファクタリングやコード生成を日常的に行う

> 💡 まずは Pro プランで始めて、「もっと使いたい」と感じたらアップグレードする。これがもっとも合理的な選び方です。

### チーム / 企業プランの場合

紹介先の企業がすでに Team プランや Enterprise プランを契約している場合は、そちらのアカウントを使ってください。企業プランには、デフォルトでデータの学習利用がオフになっている、管理者による権限管理ができるなど、実務向けの利点があります。所属企業に確認してみてください。

## 費用を抑えるコツ

Pro プランの月額 $20 は固定費ですが、以下の点を意識するとより効率的に使えます。

**1. 年払いを検討する**

月払い $20/月に対して、年払いは $200/年（$17/月相当）です。長期的に使う予定があれば年払いがお得です。

**2. 指示の粒度を意識する**

曖昧な指示で何度もやり直すより、明確な指示を1回出すほうが利用量を節約できます。これは Part 2 以降で実践しながら身につけていきます。

**3. 学習時は個人プロジェクトで練習する**

実務プロジェクトで試行錯誤するよりも、個人プロジェクトや練習用プロジェクトで操作を覚えてから実務に活用するのが効率的です。この教材もそのように設計しています。

## アカウント作成の流れ

Claude Code を使い始めるまでの流れは以下の通りです。具体的なセットアップ手順は Part 2 の Chapter 1 で画面を見ながら進めますので、ここでは全体の流れだけ把握してください。

```
1. Anthropic アカウントの作成
   claude.ai にアクセスしてアカウントを作成
       ↓
2. プランの選択と支払い情報の登録
   Pro プラン（またはその他のプラン）を選択
       ↓
3. Claude Code のインストール
   ターミナルでインストールコマンドを実行
       ↓
4. 認証（ログイン）
   ブラウザで認証を完了
       ↓
5. 初回起動
   プロジェクトディレクトリで claude コマンドを実行
```

> ⚠️ **よくあるつまずき**: アカウント作成時にメールアドレスの確認が必要です。確認メールが届かない場合は、迷惑メールフォルダを確認してください。また、支払い方法としてクレジットカードが必要です。

> 📝 すでに claude.ai のアカウント（チャット用）を持っている場合は、同じアカウントで Claude Code を利用できます。新たにアカウントを作成する必要はありません。ただし、プランが Free の場合は Pro 以上へのアップグレードが必要です。

## ✨ まとめ

- Claude Code の利用には有料プランが必要。無料プランでは使えない
- この教材の学習には **Pro プラン（$20/月）** で十分。機能の制限はない
- 年払い（$200/年）にすると月あたり $17 に。まずは月払いで始めて、継続するなら年払いに切り替えるのもあり
- 利用量が足りなくなったら Max プランにアップグレード。最初から高いプランにする必要はない
- 企業が Team / Enterprise プランを契約している場合は、そちらのアカウントを使う
- 具体的なセットアップ手順は Part 2 の Chapter 1 で詳しく解説する
