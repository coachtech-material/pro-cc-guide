# 1-1-1 なぜ Claude Code を使うのか

## 概要

この Chapter「この教材について」では、Claude Code を学ぶ意義、対象読者と前提条件、教材の全体構成を説明します。教材を読み進める土台を整える Chapter です。

この教材は、COACHTECH を優秀な成績で卒業し、選抜試験を経てフリーランスエージェントに所属しているジュニアエンジニア — Pro生の皆さんを対象としています。Claude Code を活用して皆さんの**生産性の向上**と**品質の担保**を実現することが、この教材の目的です。

最初の Section であるこの Section では、以下の順に解説します。

- なぜ今 Claude Code を学ぶ必要があるのか
- この教材で何を目指すのか
- Claude Code がどのような AI コーディングツールなのか

## AI 時代のジュニアエンジニアの現在地

### 市場の変化

AI の進化により、ソフトウェア開発の現場は大きく変わりつつあります。これまでジュニアエンジニアが担ってきた定型的なコーディング作業の一部を AI が代替できるようになり、企業はジュニアエンジニアの採用に対してより慎重になっています。

この変化は脅威であると同時に、大きなチャンスでもあります。AI を道具として使いこなせるジュニアエンジニアは、従来のジュニアレベルを超えた生産性と品質を発揮できるからです。

### 求められる力の変化

今、ジュニアエンジニアに求められているのは、AI を使いこなし、従来ミドル・シニアエンジニアが担っていた領域の一部をカバーできる力です。具体的には以下のような力が重要になっています。

- **AI への的確な指示**: 既存コードの文脈を踏まえて、AI に正確で効率的な指示を出せる
- **AI 出力の検証**: AI が生成したコードを鵜呑みにせず、自分で読み、理解し、品質を判断できる
- **継続的な学習**: AI ツールの進化に追従し、新しい機能を自ら試して実務に取り入れられる

### コーディングエージェントの進化

AI コーディングツールは、ここ数年で急速に進化してきました。

1. **補完型の時代（2021年〜）**: GitHub Copilot に代表される、エディタ上でコードの続きを提案するツールが登場しました。開発者がコードを書く速度は向上しましたが、AI の役割はあくまで「補助」でした
2. **エディタ統合型の時代（2023年〜）**: Cursor のような AI 機能を統合したエディタが登場し、チャットでコードの生成や修正を依頼できるようになりました。AI との対話がより自然になり、開発体験が変わり始めました
3. **エージェント型の時代（2025年〜）**: Claude Code のような自律的に動作するエージェントが登場しました。コードベース全体を読み取り、ファイルを編集し、コマンドを実行し、外部ツールと連携する — AI が「補助」から「協働パートナー」へと進化しています

この教材では、エージェント型の代表的なツールである Claude Code の活用方法を学びます。

## この教材で目指すこと

### 生産性の向上と品質の担保

この教材の柱は2つあります。

- **生産性の向上**: Claude Code を使いこなすことで、コーディング、テスト作成、バグ修正、リファクタリングといった作業を高速に進められるようになります
- **品質の担保**: AI が生成したコードを自分で検証し、品質と安全性に責任を持てるようになります

この2つは車の両輪です。速く作れても品質が低ければ意味がありませんし、品質にこだわっても速度が出なければ実務では評価されません。Claude Code を正しく活用することで、この両立を目指します。

### 3つの能力

この教材を通じて、以下の3つの能力を養います。

| 能力 | 概要 |
|---|---|
| **使いこなす力（Input）** | Claude Code の機能を理解し、既存コードの文脈を踏まえて的確な指示を設計することで、開発の生産性を高める力 |
| **見極める力（Output）** | Claude Code の出力を検証・判断し、その判断を自分の言葉で説明できる力。品質と安全性に責任を持つ |
| **学び続ける力（Time）** | ツール・技術の進化を追い、自ら試し、実務に適応し続ける力 |

特に「見極める力」は重要です。AI の出力をそのまま受け入れる「思考の放棄」に陥らず、自分で理解・判断する習慣を保つことが、プロのエンジニアとしての責任です。

### 実務タスクとコードリーディング

この教材では、以下の実務タスクを Claude Code と協働して遂行する力を身につけます。

- **バグ修正**: 既存コードの不具合を特定し、修正する
- **機能開発**: 新規機能の実装や既存機能の改修を行う
- **リファクタリング**: コードの品質・可読性・パフォーマンスを改善する

これらのタスクはすべて、**コードリーディング**を前提としています。既存プロジェクトのコーディング規約と設計方針を理解した上で、それに従いながら作業を進めます。ゼロからコードを書くのではなく、既存のコードベースの中で働く — これが実務の現実です。

## Claude Code の特徴

### AI コーディングツールのアプローチの違い

AI コーディングツールは、大きく2つのアプローチに分けられます。

**補完型（Code Completion）**

エディタ上でコードの続きを予測・提案するアプローチです。GitHub Copilot が代表例で、開発者が書いているコードの文脈から次に書くべきコードを提案します。開発者がコードを書く主体であり、AI はリアルタイムに補助する役割です。

**エージェント型（Agentic Coding）**

自然言語の指示を受けて、AI が自律的にタスクを遂行するアプローチです。Claude Code はこのエージェント型に分類されます。エージェント型ツールは以下のような動作を自律的に行います。

- **コードベースの読み取り**: プロジェクト全体の構造を理解し、関連するファイルを見つけ出す
- **ファイルの編集**: 複数のファイルにまたがる変更を一度に行う
- **コマンドの実行**: テストの実行、ビルド、Git 操作などをターミナルで実行する
- **ツールとの統合**: MCP（Model Context Protocol）を通じて外部ツール（データベース、API、プロジェクト管理ツールなど）と連携する

補完型が「一行ずつ手伝う」のに対し、エージェント型は「タスク全体を一緒に進める」という違いがあります。

### Claude Code の特徴と強み

Claude Code は Anthropic が提供するエージェント型の AI コーディングツールです。

**コードベース全体の理解**

Claude Code はプロジェクトのファイル構造、コードの依存関係、設定ファイルを読み取り、コードベース全体を理解した上で作業を行います。「この関数を修正して」という指示に対して、関連する他のファイルへの影響も考慮した修正を提案できます。

**自律的なタスク遂行**

指示を受けると、Claude Code は計画を立て、コードを書き、テストを実行し、結果を確認するというサイクルを自律的に回します。エラーが発生した場合も、自分で原因を特定して修正を試みます。

**ターミナルネイティブ**

Claude Code はターミナルで動作するコマンドラインツールです。開発者が日常的に使うターミナル環境にそのまま統合され、ファイル操作・Git 操作・テスト実行といった開発作業をシームレスに行えます。

**利用環境の多様性**

Claude Code はターミナル（CLI）を基盤としながら、複数の環境から利用できます。

- **ターミナル（CLI）**: フル機能の CLI。ファイル編集、コマンド実行、プロジェクト管理をコマンドラインから行えます
- **VS Code / Cursor**: エディタ内でインラインの diff 表示、ファイル参照、会話履歴の管理ができます
- **JetBrains IDE**: IntelliJ IDEA、PyCharm、WebStorm などの JetBrains 製 IDE で利用できます
- **デスクトップアプリ**: IDE やターミナルの外で Claude Code を使える独立したアプリです
- **Web**: ブラウザ上でローカル環境のセットアップなしに利用できます

この教材ではターミナル（CLI）を主な利用環境としますが、VS Code のターミナル内での利用も想定しています。

**カスタマイズと拡張**

`CLAUDE.md` ファイルによるプロジェクト固有の指示、Skills によるワークフローの定型化、Hooks によるアクションの自動化、MCP による外部ツール連携など、開発チームのワークフローに合わせた柔軟なカスタマイズが可能です。

## まとめ

- AI の進化により、ジュニアエンジニアには AI を使いこなし、出力を検証し、継続的に学ぶ力が求められている
- この教材は「生産性の向上」と「品質の担保」を両輪とし、使いこなす力・見極める力・学び続ける力の3つの能力を養う
- 実務タスク（バグ修正・機能開発・リファクタリング）をコードリーディングを前提に Claude Code と協働して遂行する
- Claude Code はエージェント型の AI コーディングツールで、コードベースの読み取り・ファイル編集・コマンド実行・ツール統合を自律的に行う
