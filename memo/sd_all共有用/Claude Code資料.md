# Claude Code 教材 概要資料

# 教材の概要

CoachTech を優秀な成績で卒業した Pro 生が、Claude Code を活用して実務の Laravel 開発で活躍できるよう支援する教材です。

Claude Code の機能を正しく使いこなし、AI の出力を自分で検証・判断できる Pro 生を育成することで、**従来ミドル層が担っていた開発タスク（バグ修正・機能開発・リファクタリング）の一部を、ジュニアの段階からカバーできるようになる**ことを目指します。

---

# 対象読者（ペルソナ）

スクールを優秀な成績で卒業し、選抜試験を経てフリーランスエージェントに所属している受講生（Pro 生）。企業に紹介され、副業・フリーランス・転職などの形で Laravel を使った実務開発に携わります。

## 前提となる技術スキル

| 分野 | レベル |
|---|---|
| ターミナル操作 | 基本コマンド（cd, ls, mkdir 等）を使ったファイル操作ができる |
| Git/GitHub | ブランチ運用、PR 作成、コンフリクト解決、コードレビューができる |
| Docker | Docker Compose を使った開発環境の構築・管理ができる |
| HTML/CSS | レスポンシブデザインを含む Web ページを作成できる |
| PHP | 基本文法、関数、OOP の基礎を理解している |
| Laravel | MVC、Eloquent ORM、認証・認可、バリデーション、RESTful API、PHPUnit を実装できる |
| データベース | RDB 設計（正規化、ER 図）、SQL（JOIN、サブクエリ、集計）を扱える |
| 開発プロセス | 要件定義 → 設計 → 実装 → テストの一連のフローを経験している |
| AI 開発ツール | 使用経験は問わない |

## 前提環境

- OS: macOS および Windows（WSL 経由）
- エディタ: VS Code
- Laravel 10
- Claude アカウント: Pro（$20/月）プランが最低条件、Max プランを推奨
- Node.js は不要（Claude Code はネイティブインストールが推奨）

# コンセプト

## ゴール

Claude Code を活用して Pro 生の**生産性の向上**と**品質の担保**を実現します。AI を使いこなすジュニアエンジニアが、従来ミドル層が担っていた領域の一部をカバーできるようになることで、市場での競争力が大きく変わります。

具体的には、**コードリーディング**を前提に、既存プロジェクトのコーディング規約と設計方針に従いながら、以下の開発タスクを Claude Code と協働して高速かつ高品質に遂行できるようになることを目指します。

- **バグ修正**: 既存コードの不具合を特定し、修正する
- **機能開発**: 新規機能の実装や既存機能の改修を行う
- **リファクタリング**: コードの品質・可読性・パフォーマンスを改善する

## 養う3つの能力

これらのタスクを Claude Code と協働して進めるために、以下の能力を養います。

| 能力 | 概要 |
|---|---|
| **1. 使いこなす力（Input）** | Claude Code の機能を理解し、既存コードの文脈を踏まえて的確な指示を設計することで、開発の生産性を高める力 |
| **2. 見極める力（Output）** | AI の出力を検証・判断し、その判断を自分の言葉で説明できる力。**「コードレビュー」「テスト」「セキュリティチェック」** を通じて品質と安全性に対する **「説明責任」** を持つ。AI の出力をそのまま受け入れる **「思考の放棄」** に陥らず、自分で理解・判断する習慣を保つことが重要 |
| **3. 学び続ける力（Time）** | ツール・技術の進化を追い、実務に適応し続ける力 |

# 教材の全体構成

教材は Part 0 〜 Part 4 の5部構成を想定しています。初期リリースでは **Part 0・1（Claude Code の基礎的な主要機能）** を提供し、**Part 2〜4 は構想段階です。** なお、Part 0・1 の構成もまだ仮のものであり、変更される可能性があります。

| Part | タイトル | 対応する能力 | 推定学習時間 | ステータス |
|---|---|---|---|---|
| **Part 0** | **はじめに** | - | 0.5 時間 | 初期リリース |
| **Part 1** | **Claude Code の基礎** | 使いこなす力 | 7 - 10 時間 | 初期リリース |
| Part 2 | Claude Code の実践 | 使いこなす力 + 見極める力 | 未定 | 構想中 |
| Part 3 | Claude Code の応用 | 使いこなす力 | 未定 | 構想中 |
| Part 4 | 継続的な学習 | 学び続ける力 | 未定 | 構想中 |
